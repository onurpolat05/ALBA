Start the interactive setup process to create a personalized AI agent system.

Read the full setup guide at `setup/initial-setup.md` and follow these steps:

1. **Discovery Phase**: Ask the user 5-7 questions about their role, workflow, tools, and preferences
2. **Structure Creation**: Based on answers, create the appropriate folder structure and files
3. **CLAUDE.md Generation**: Create a personalized CLAUDE.md tailored to their needs
4. **Initial Content**: Populate memory files with their current tasks and priorities
5. **Test & Confirm**: Run a test to ensure everything works

Important guidelines:
- Ask questions ONE at a time using AskUserQuestion tool
- Adapt complexity based on their technical level
- Create only what's needed (minimal by default)
- Always explain what you're creating and why
- Test with `/start` command at the end

Begin by greeting the user and asking about their primary role.
