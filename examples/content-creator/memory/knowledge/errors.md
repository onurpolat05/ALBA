# Error Solutions

Record of solved errors and their solutions. Reference when similar issues occur.

## Format

```
### [Error Type/Message]
**When:** [Context]
**Solution:** [What fixed it]
**Prevention:** [How to avoid]
```

---

## Errors

### Newsletter Sent With Broken Links
**When:** Newsletter #43 - two links pointed to draft URLs instead of published posts.
**Solution:** Manually sent correction email with working links.
**Prevention:** Always test every link in preview mode before sending. Add "link check" as a mandatory step in newsletter workflow.

### Blog Post Published Without Meta Description
**When:** "AI Image Tools" post - SEO meta was left blank.
**Solution:** Added meta description post-publish, but lost initial SEO indexing window.
**Prevention:** Add meta description to the publishing checklist. Draft it during the outline phase, not at the end.
