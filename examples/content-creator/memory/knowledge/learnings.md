## Entries

### LinkedIn Algorithm Favors Native Content
**Context:** Noticed external links get 40% fewer impressions than text-only posts.
**Insight:** LinkedIn deprioritizes posts with external links. Put the link in comments instead.
**Application:** Always post value-first text, add blog/newsletter links in the first comment.

### Newsletter Subject Lines With Numbers Perform Better
**Context:** A/B tested subject lines over 8 weeks.
**Insight:** Subject lines with specific numbers ("5 tools", "3 lessons") had 15% higher open rates than generic ones.
**Application:** Include a number in newsletter subject lines when possible. Be specific.

### X Threads Need a Standalone Hook Tweet
**Context:** Thread engagement dropped when hook tweet required context from thread body.
**Insight:** The first tweet must provide value or intrigue on its own - many people only see tweet #1.
**Application:** Write the hook tweet last, after the thread is done. Test: does it work as a standalone post?
