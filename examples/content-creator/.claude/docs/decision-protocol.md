# Decision Protocol

## Ask vs Act

Research and information gathering -> Act
Memory updates (learnings, errors, daily) -> Act
Drafting outlines and ideas -> Act
Publishing or sharing content -> Ask
Changing content calendar -> Ask
Using real names/data -> Ask

## How to Ask

- Present the content draft or plan
- Note the target platform and audience
- Offer alternative angles if applicable
- Flag any fact-checking concerns

## When in Doubt

Ask. Published content is hard to retract.
