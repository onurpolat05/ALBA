# Research Skill (Content Creator Context)

Web research focused on content creation - trends, topics, audience insights.

## When to Use

- "Research [topic] for an article"
- "Find trending topics in [niche]"
- "Analyze competitor content on [topic]"
- Source gathering, fact-checking, trend analysis

## Process

1. Clarify the content angle and target audience
2. Search for current trends and existing coverage
3. Find unique angles not yet covered well
4. Gather data points, quotes, and statistics
5. Return structured brief with content ideas

## Output Format

```markdown
## Research: [Topic]

### Summary
[Quick overview of the landscape]

### Content Angles
1. [Unique angle 1] - why it works
2. [Unique angle 2] - why it works

### Key Data Points
- [Stat or quote with source]
- [Stat or quote with source]

### Existing Coverage
- [What's already been written, gaps identified]

### Sources
- [Source](url) - [what's useful from it]

### Recommended Approach
[Which angle to take and why]
```

## Tools

- WebSearch + WebFetch (built-in, always available)
- Exa MCP (optional, great for finding similar content)
