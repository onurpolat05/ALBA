# Research Preferences

## Citation Format
- APA 7th edition for papers
- Inline links for informal notes
- Always include DOI when available

## Source Reliability
- Peer-reviewed > preprint > blog > social media
- Prefer sources < 2 years old for fast-moving fields
- Cross-reference claims across 2+ independent sources

## Output Preferences
- Tables for comparisons
- Bullet points for findings
- Confidence ratings on all claims
- "Gaps" section in every research output
