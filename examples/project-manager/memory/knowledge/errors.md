# Error Solutions

Record of solved errors and their solutions. Reference when similar issues occur.

## Format

```
### [Error Type/Message]
**When:** [Context]
**Solution:** [What fixed it]
**Prevention:** [How to avoid]
```

---

## Errors

### Missed Stakeholder in Sprint Review Invite
**When:** Sprint 2 review - VP of Product was not invited
**Solution:** Sent last-minute invite, VP attended late. Added VP to permanent invite list.
**Prevention:** Maintain a stakeholder distribution list. Review attendee list against it before every sprint event.

### Trello Board Sync Conflict
**When:** Two team members moved the same card simultaneously, lost comment history
**Solution:** Restored card from Trello activity log. Established rule: only card owner moves cards between lists.
**Prevention:** Define card ownership. Use Trello automation (Butler) to restrict list transitions to assigned members.
