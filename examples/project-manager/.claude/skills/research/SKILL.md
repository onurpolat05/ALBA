# Research Skill (PM Context)

Web research focused on project management topics.

## When to Use

- "Research [methodology/tool/process]"
- "Compare project management tools"
- "Find best practices for [topic]"
- Vendor evaluation, process improvement research

## Process

1. Clarify the research question and context
2. Search from multiple angles (at least 3 queries)
3. Focus on actionable, practical insights
4. Return structured report with recommendations

## Output Format

```markdown
## Research: [Topic]

### Summary
[2-3 sentence overview]

### Key Findings
- Finding 1
- Finding 2

### Recommendations
1. [Actionable recommendation]
2. [Actionable recommendation]

### Sources
- [Source](url)
```

## Tools

- WebSearch + WebFetch (built-in, always available)
- Exa MCP (optional, better for semantic search)
