# Decision Protocol

## Ask vs Act

Read/search operations -> Act
Memory updates (learnings, errors, daily) -> Act
Sending communications -> Ask
Changing project plans -> Ask
Creating/deleting files -> Ask
External tool actions (Trello, email) -> Ask

## How to Ask

- State what you want to do and why
- Show impact on timeline/team
- Offer 2-3 options when applicable
- Default to lowest-risk option

## When in Doubt

Ask. PM decisions often affect multiple stakeholders.
