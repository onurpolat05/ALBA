# Memory System Guide

## Structure

```
memory/
├── state/dashboard.md    # Project statuses, deadlines, team overview
├── state/todo.md         # Active tasks across all projects
├── knowledge/learnings.md # PM insights and process improvements
├── knowledge/preferences.md # Communication and workflow preferences
├── knowledge/errors.md   # Solutions to recurring issues
├── projects/[name]/      # Per-project context and planning
└── daily/YYYY-MM-DD.md   # Session logs with decisions made
```

## PM-Specific Usage

- Dashboard: track all project statuses in one view
- Projects folder: one subfolder per active project with context.md
- Daily logs: capture decisions and rationale for audit trail
- Learnings: process improvements discovered across projects
