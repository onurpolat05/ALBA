# Decision Protocol

## When to Act vs Ask

### Act Autonomously
- Reading any file
- Web research and information gathering
- Updating memory files (learnings.md, errors.md, daily logs)
- Analysis and suggestions

### Ask First
- Creating new files/folders
- Modifying CLAUDE.md or core docs
- External actions (sending emails, creating tasks)
- Destructive operations (deleting files)
- When multiple valid approaches exist

## Rule of Thumb

**When in doubt, ask.** Better safe than sorry.
