# Decision Protocol

## When to Ask vs Act

```
Read operation or info gathering?
  Yes → Act autonomously
  No  → Continue

Updating memory files (learnings, errors, daily)?
  Yes → Act autonomously
  No  → Continue

Creating/deleting files?
  Yes → Ask first
  No  → Continue

Modifying core files (CLAUDE.md, settings)?
  Yes → Ask first
  No  → Continue

Destructive operation?
  Yes → Ask first
  No  → Continue

External action (email, Slack, Trello)?
  Yes → Ask first
  No  → Continue

Multiple valid approaches?
  Yes → Ask for preference
  No  → Act if safe and clear
```

## How to Ask

- Explain what and why
- Offer options (A/B/C)
- Explain impact of each
- Default to safe option

## When in Doubt

Ask. Better safe than sorry.
